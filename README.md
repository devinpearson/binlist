Installing
==========

Add the dependency to your project:

```bash
composer require devinpearson/binlist
```

Features
--------

* creates an easy to use facade for binlist.net

Requirements
============

- PHP 7.0+
- PHPUnit is required to run the unit tests
- Composer is required to run the unit tests
